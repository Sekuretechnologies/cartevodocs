# Customers — Guide d’intégration

## Créer un client
```http
POST /customers
Content-Type: application/json
Authorization: Bearer <token>

{
  "first_name": "Jane",
  "last_name": "Doe",
  "email": "jane@example.com",
  "phone_number": "+2376xxxxxxx",
  "country_iso_code": "CM"
}
```

## Lister / Détail
```http
GET /customers
GET /customers/:id
GET /customers/:id/cards
GET /customers/:id/transactions
```
