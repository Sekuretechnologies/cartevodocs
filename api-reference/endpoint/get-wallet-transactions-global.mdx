# Get Transactions

Retourne toutes les transactions, filtrables par statut et type, avec pagination.

## Authorizations

- Authorization: Bearer `<token>` (required)

## Query Parameters

- status: SUCCESS | PENDING | FAILED
- type: FUND | DEPOSIT | TRANSFER | COLLECTION
- page, limit
