# Get Transaction

Retourne le détail d’une transaction par identifiant. Idéal pour vérifier l’état final d’une opération et en extraire les informations de rapprochement.

Cas d’usage

- Vérifier le statut final (SUCCESS/PENDING/FAILED)
- Afficher le montant, la devise et les métadonnées associées

## Try it

<ApiPlayground schema="/openapi.json" endpoint="/wallet/transactions/{id}" method="get" />

## Authorizations

- Authorization: Bearer `<token>` (required)

## Path Parameters

- id (string) — transaction identifier (required)

## Notes

- Utilisez cet endpoint pour vérifier la source, le statut et les montants
- Rafraîchissez l’historique après confirmation
