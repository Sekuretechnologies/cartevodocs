# Get Wallet Transactions

Retourne les transactions liées à un portefeuille. Servez‑vous de cette route pour afficher l’historique, filtrer par période côté client et déclencher un rafraîchissement après une opération (fund/deposit/transfer).

Cas d’usage

- Journaliser toutes les opérations d’un portefeuille
- Confirmer visuellement la réussite d’un encaissement/dépôt

## Try it

<ApiPlayground schema="/openapi.json" endpoint="/wallet/transactions/wallet/{walletId}" method="get" />

## Authorizations

- Authorization: Bearer `<token>` (required)

## Path Parameters

- walletId (string) — wallet identifier (required)

## Query Parameters

- page, limit — pagination

## Notes

- Filtrez par statut/type via l’endpoint global si nécessaire
- Combinez avec `GET /wallet/transactions/{id}` pour un affichage détaillé
