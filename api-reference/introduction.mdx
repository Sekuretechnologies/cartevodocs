# Référence API — Cartevo

Cette section décrit les endpoints utilisés par les entreprises pour exploiter Cartevo. Les exemples omettent toute information sensible.

## Bases

- Authentification: `Authorization: Bearer <token>`
- Serveur (prod): `https://api.cartevo.co/api/v1`
- Formats: JSON uniquement
- Idempotence: utilisez un en‑tête `Idempotency-Key` pour les opérations sensibles

## Tester depuis la doc

Les pages d’API incluent des exemples cURL/JS. Pour des tests interactifs, référez‑vous à l’OpenAPI intégré et au Playground Mintlify.
