## À propos

Cartevo permet aux entreprises de:

- Gérer des portefeuilles (création, détail, solde)
- Alimenter des portefeuilles (alimentation locale, dépôt avec change, crédit test en pré‑production)
- Effectuer des transferts (internes et entre portefeuilles)
- Consulter les transactions (par portefeuille, client, statut)
- Gérer les clients (création, liste, détails)
- Convertir des montants via un endpoint dédié (les taux sont gérés par l’administrateur)

### Capacités clés

- Création multi‑devises: XAF, USD, …
- Encodage des opérations: idempotence, pagination, filtres par statut/type
- Sécurité: authentification Bearer, séparation pré‑production/production
- Observabilité: `request-id`, webhooks (collections)

## Exemples rapides

Créer un portefeuille (cURL)
```bash
curl -X POST "$BASE_URL/wallets" \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "currency": "USD",
    "country_iso_code": "US"
  }'
```

Lister les transactions d’un portefeuille (Node.js)
```javascript
import fetch from "node-fetch";

const res = await fetch(`${process.env.BASE_URL}/wallet/transactions/wallet/${process.env.WALLET_ID}`, {
  headers: { Authorization: `Bearer ${process.env.API_KEY}` }
});
const data = await res.json();
```
